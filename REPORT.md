# Kinematic Measurement Console

## 1. 프로젝트 개요

본 프로젝트는 카메라 영상에서 색상 마커를 추적하고, Kalman Filter를 이용해 물체의 위치, 속도, 가속도 등 물리량을 실시간으로 추정하는 실험 도구이다. 초기 구현은 단진자 실험을 중심으로 시작했지만, 현재는 자유낙하, 포물선 운동, 직선/수레 운동, 용수철 진동, 원운동까지 확장 가능한 구조로 개선하였다.

핵심 목표는 다음과 같다.

- 별도 센서 없이 카메라 영상만으로 물체 운동을 측정한다.
- noisy한 위치 데이터에서 속도와 가속도를 안정적으로 추정한다.
- 실험별 물리량을 CSV와 실시간 그래프로 제공한다.
- 단진자에서는 지배 방정식 형태를 데이터에서 추정한다.

## 2. 개발 동기

일반적인 물리 실험은 사용자가 이미 알고 있는 공식을 검증하는 방식으로 진행된다. 예를 들어 단진자에서는 `T = 2π√(L/g)` 또는 `θ'' = -(g/L)sinθ`를 전제로 두고 실험값을 비교한다.

본 프로젝트는 반대로, 영상에서 얻은 측정 데이터만으로 운동 상태를 추정하고 물리량을 계산하는 방향을 택했다. 즉, 카메라 기반 추적과 Kalman Filter를 결합해 여러 실험에서 공통적으로 사용할 수 있는 측정 플랫폼을 만드는 것이 목적이다.

## 3. 시스템 구성

시스템은 크게 네 단계로 동작한다.

1. 카메라 영상 입력
2. 색상 마커 추적
3. Kalman Filter 기반 상태 추정
4. 실험별 물리량 분석 및 결과 출력

전체 흐름은 다음과 같다.

```text
camera frame
  -> HSV color tracking
  -> marker position (pixel)
  -> calibration: pixel -> meter or angle
  -> Kalman Filter
  -> position / velocity / acceleration
  -> experiment-specific analysis
  -> graph + readout + CSV
```

## 4. 주요 기능

### 4.1 색상 마커 추적

마커 색상을 HSV 범위로 분리하고, 연결된 blob 중 가장 적절한 후보를 선택한다. 단순히 같은 색 픽셀의 평균을 내는 방식은 입술, 옷, 배경 등 비슷한 색 영역에 끌릴 수 있기 때문에, 현재는 다음 조건을 함께 사용한다.

- 이전 프레임 위치와 가까운 blob 우선
- 너무 길쭉한 blob 제외
- 사용자가 클릭한 마커 색상 자동 샘플링
- 추적 위치 smoothing
- 추적 실패 시 일정 시간 동안 재획득 제한

특히 V자 진자 보정 마지막 단계에서 추 중심을 클릭하면, 그 지점이 마커 위치와 색상 샘플로 동시에 사용된다.

### 4.2 Kalman Filter

2D 운동 실험에서는 다음 상태를 추정한다.

```text
state = [x, y, vx, vy, ax, ay]
```

카메라에서 직접 얻는 값은 위치뿐이지만, Kalman Filter를 통해 속도와 가속도를 함께 추정한다. 단순 미분 방식은 노이즈가 증폭되어 가속도가 크게 흔들리므로, 필터 기반 추정이 필요하다.

### 4.3 V자 단진자 보정

실제 실험 장치가 한 점 고정이 아니라 V자 형태의 두 고정점으로 구성되어 있으므로, 일반적인 단일 pivot 보정은 부정확하다. 이를 해결하기 위해 다음 절차를 적용했다.

1. 왼쪽 상단 고정점 클릭
2. 오른쪽 상단 고정점 클릭
3. 정지 상태의 추 중심 클릭

앱은 두 고정점의 중점을 `PIVOT`으로 사용하고, 실 한 가닥 길이와 V자 반폭을 이용해 유효 진자 길이 `L_eff`를 계산한다.

```text
L_eff = l * sqrt(l_px^2 - a_px^2) / l_px
```

여기서 `l`은 실제 실 한 가닥 길이, `l_px`는 화면에서 측정한 평균 실 길이, `a_px`는 두 상단 고정점 사이 거리의 절반이다.

## 5. 지원 실험

현재 웹 UI에서 지원하는 실험 프리셋은 다음과 같다.

| 실험 | 측정 및 분석 물리량 |
|---|---|
| Pendulum EKF | `theta`, `omega`, `alpha`, `gamma`, `L_eff` |
| Free Fall | `y`, `vy`, `ay`, `g_est`, `g_error` |
| Projectile | `x`, `y`, `speed`, `range`, `height`, `mean ay` |
| Linear Cart | `x`, `vx`, `ax`, `mean ax`, `F=ma` |
| Spring-Mass | `x`, `vx`, `ax`, `T`, `omega`, `k=mω²` |
| Circular Motion | `r`, `speed`, `v²/r`, measured `|a|` |
| Generic 2D | `x`, `y`, `vx`, `vy`, `ax`, `ay`, `force` |

## 6. 대표 실험 1: V자 단진자

단진자 실험에서는 마커의 위치를 각도 `theta`로 변환한 뒤, 비선형 진자 모델에 대한 Extended Kalman Filter를 적용한다.

사용한 운동 모델은 다음과 같다.

```text
theta_dot = omega
omega_dot = -(g / L) sin(theta) - gamma * omega
```

여기서 `gamma`는 감쇠계수이고, 앱은 이를 확장 상태로 포함해 추정할 수 있다.

단진자 모드에서는 다음 값이 실시간으로 표시된다.

- 각도 `theta`
- 각속도 `omega`
- 각가속도 `alpha`
- 감쇠계수 `gamma`
- 유효 진자 길이 `L_eff`
- 위상공간 그래프

또한 충분한 데이터가 쌓이면 STLSQ 기반으로 다음 형태의 방정식을 추정한다.

```text
theta_ddot = c1 * omega + c2 * sin(theta)
```

이 결과를 통해 실험 데이터에서 `sin(theta)` 항과 감쇠 항이 실제로 선택되는지 확인할 수 있다.

## 7. 대표 실험 2: 자유낙하

자유낙하 실험은 본 도구가 진자 전용이 아니라 일반적인 직선 운동도 측정할 수 있음을 보여주기 위해 추가하였다.

자유낙하 모드에서는 진자 보정을 사용하지 않는다. 대신 화면에서 기준 거리 보정을 수행한다.

절차는 다음과 같다.

1. 실험 프리셋을 `Free Fall`로 선택
2. `Scale reference`에 기준 거리 입력
   - 30 cm 자 사용 시 `0.30`
   - 50 cm 기준선 사용 시 `0.50`
3. `Sample Color`로 마커 색상 샘플링
4. `Calibrate` 클릭
5. 원점 클릭
6. 기준 거리 끝점 클릭
7. 물체를 던지지 않고 그대로 놓아 낙하시킴

자유낙하 모드에서 표시되는 값은 다음과 같다.

```text
y       수직 위치
vy      수직 속도
ay      수직 가속도
g_est   중력가속도 추정값
g_error 9.81 m/s^2 대비 오차율
```

실험에서는 공기저항을 주요 모델에 포함하지 않았다. 낙하 거리가 짧고 물체가 작고 충분히 무거운 경우, 공기저항보다 카메라 fps, 기준 거리 보정, 마커 추적 오차가 더 큰 영향을 준다.

오차 요인은 다음과 같다.

- 카메라 fps 한계
- 기준 거리 보정 오차
- 손을 놓는 순간의 초기 속도
- 마커 추적 오차
- 카메라와 낙하면의 각도 차이
- 공기저항

## 8. 대표 실험 3: 직선/수레 운동

직선/수레 운동 실험은 물체의 1차원 이동을 추적하고, 질량 입력값을 이용해 힘을 계산하는 실험이다.

측정값은 다음과 같다.

```text
x       위치
vx      속도
ax      가속도
mean ax 평균 가속도
F=ma    힘 추정값
```

이 실험은 손으로 수레를 밀거나, 낮은 경사면에서 수레를 굴리는 방식으로 진행할 수 있다. 경사면을 사용하면 손으로 미는 방식보다 가속도가 비교적 일정해져 결과가 안정적이다.

## 9. CSV 저장 및 후속 분석

앱은 실험 데이터를 CSV로 저장한다. 기존에는 공통 `primary`, `velocity`, `acceleration` 위주로 저장했지만, 현재는 다른 실험 분석을 위해 다음 확장 컬럼도 함께 저장한다.

- `experiment`
- `x_kalman`, `y_kalman`
- `vx_kalman`, `vy_kalman`
- `ax_kalman`, `ay_kalman`
- `speed`
- `force`
- `primary_measured_y`
- `velocity_y_numerical`
- `acceleration_y_numerical`

이를 통해 자유낙하, 포물선, 수레 운동 데이터를 엑셀이나 Python으로 추가 분석할 수 있다.

## 11. 한계

현재 시스템의 한계는 다음과 같다.

1. 단일 카메라 기반이므로 깊이 방향 운동은 정확히 측정하기 어렵다.
2. 카메라와 운동 평면이 평행하지 않으면 길이와 각도 오차가 발생한다.
3. 마커 색상과 배경 색상이 비슷하면 오탐이 발생할 수 있다.
4. 자유낙하처럼 빠른 실험은 낮은 fps에서 오차가 커질 수 있다.
5. 단진자 방정식 추정은 현재 진자 모드에만 적용된다.
6. 다른 실험은 방정식 자동 발견보다는 물리량 측정과 CSV 분석 중심이다.

## 12. 향후 개선 방향

향후 개선할 수 있는 부분은 다음과 같다.

- 자유낙하에서 낙하 시작/충돌 이후 구간 자동 제거
- 포물선 운동에서 2차 곡선 피팅 추가
- 용수철 운동에서 피크 검출 안정화 및 감쇠계수 추정
- 원운동에서 중심 자동 추정
- 마커 추적에 색상뿐 아니라 원형도, 크기, 템플릿 매칭 추가
- 모바일 카메라 연결 UX 개선
- 실험별 보고서 자동 생성

## 13. 결론

본 프로젝트는 카메라 영상만으로 물체의 운동을 추적하고, Kalman Filter를 통해 위치, 속도, 가속도를 안정적으로 추정하는 실험 도구를 구현하였다. 단진자 실험에서는 V자 고정 구조를 반영한 보정과 방정식 추정을 지원하고, 자유낙하 및 직선 운동에서는 일반적인 병진 운동의 물리량을 측정할 수 있도록 확장하였다.

따라서 이 시스템은 단순한 진자 전용 프로그램이 아니라, 여러 물리 실험에서 공통적으로 사용할 수 있는 영상 기반 물리량 측정 플랫폼으로 볼 수 있다.
