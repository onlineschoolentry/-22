# PhysicsLens Lab Web

브라우저에서 카메라 기반 물리 실험을 실행하는 UI입니다. 단진자뿐 아니라 자유낙하, 포물선 운동, 직선 운동, 용수철 진동, 원운동의 공통 물리량을 측정합니다.

## 실행

```bash
cd C:\Users\이성현\Downloads\physics_kalman\physics_kalman
python web_server.py
```

서버가 출력하는 주소로 접속합니다.

- PC: `https://127.0.0.1:8843/`
- 휴대폰: `https://<PC-LAN-IP>:8843/`

휴대폰 카메라는 HTTPS가 필요합니다. 인증서 경고가 나오면 고급/세부정보에서 계속 진행하세요.

## 실험 순서

1. 실험 프리셋을 선택합니다.
2. 마커 색상을 선택하거나 `색상 샘플`로 마커를 직접 클릭합니다.
3. `카메라 시작`을 누릅니다.
4. `보정`을 누르고 화면의 안내대로 기준점을 클릭합니다.
5. 실험을 진행하면서 위치, 속도, 가속도, 힘 그래프를 확인합니다.
6. `CSV 저장`으로 시계열 데이터를 저장합니다.
7. 진자 프리셋에서는 `방정식 추정`으로 STLSQ 결과를 확인합니다.

## V자 진자 보정

사진처럼 실이 V자로 고정된 진자는 다음 순서로 보정합니다.

1. `Pendulum length (m)`에 실 한 가닥의 실제 길이를 입력합니다.
2. `보정`을 누릅니다.
3. 왼쪽 상단 고정점을 클릭합니다.
4. 오른쪽 상단 고정점을 클릭합니다.
5. 정지 상태의 추 중심을 클릭합니다.

앱은 두 상단 고정점의 중점을 `PIVOT`으로 표시하고, V자 반폭을 반영한 유효 길이 `L_eff`를 계산해 진자 EKF에 사용합니다. 추적 중에는 두 고정점에서 추까지의 V자 실이 화면에 표시됩니다.

## 2D 프리셋 보정

| 프리셋 | 보정 방식 |
|---|---|
| 자유낙하/포물선/직선/용수철/원운동/범용 2D | 원점 또는 평형점 클릭 후 기준 거리 끝점 클릭 |

## 구현 범위

- 브라우저 카메라 입력
- HSV 색상 마커 추적
- V자 진자 보정 및 유효 길이 계산
- 단진자 EKF
- 범용 2D Motion Kalman Filter
- 실시간 위치/속도/가속도/위상 그래프
- 힘 및 중력가속도 추정용 리드아웃
- CSV 다운로드
- 진자 전용 RTS smoothing + STLSQ 방정식 추정 API
